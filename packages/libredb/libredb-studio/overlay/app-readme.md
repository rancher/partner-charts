# LibreDB Studio

LibreDB Studio is an MIT-licensed, AI-powered open source SQL IDE that connects to PostgreSQL, MySQL, Oracle, SQL Server, SQLite, MongoDB and Redis directly from the browser.

Use cases:

* Browser-based database management
  * Connect to and query PostgreSQL, MySQL, Oracle, SQL Server, SQLite, MongoDB and Redis from a single web interface, with no separate desktop client to install.
* AI-assisted querying
  * An optional AI assistant (Gemini, OpenAI, or a local model you provide the key for) can write and explain SQL from natural language. It stays off unless you configure a provider.
* Self-hosted data
  * Runs entirely on your own infrastructure with SQLite storage by default, so no external database is required to operate the IDE itself.
