# HariKube
[HariKube](https://harikube.info/) is a Kubernetes hyper-scaler that turns your cluster into a zero-effort Platform-as-a-Service. By leveraging a Unified Service Model, it removes technical bottlenecks to let your services run as native Kube citizens.

> If you use any other namespace then `harikube`, please set `vcluster.exportKubeConfig.server` to `https://harikube.<NAMESPACE>:443`!!!
